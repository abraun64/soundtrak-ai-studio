# Specs — index by category

The files in this folder are the system's **canonical contracts**. When an artifact doesn't
look right, check its spec first. They fall into three kinds — open the one you need below,
or see the [full navigation index](../NAVIGATION_INDEX.md#2-specs-canonical-schemas--all-campaigns-inherit).

> This is an index, not a spec. The files themselves stay flat in `docs/specs/` (everything
> references them by path); this README is the categorised view (SYS-024, option A).
