## Strategic spine
- **Audience**: advisers navigating regulatory, tech + client-expectation change.
- **Insight**: disruption read as threat paralyses; read as opportunity, it mobilises.
- **Big idea**: a named franchise — "Change/Chance" — that frames industry upheaval as the chance to think differently, spanning four pillars (technology, sales/marketing, innovation, adviser solutions).
- **Channel**: six+ downloadable guides + webinars + podcasts + video.
- **Ran**: a ~3-month multi-step campaign (time-boxed burst).

## Creative move
**What it did**: built a durable franchise on a single reframe.
**The move**:
- Reframe threat → opportunity ("your chance to think differently") as the franchise spine.
- Four pillars → many entry points for different adviser pain points (AFSL licensing, dealer-group choice, tech selection).
- Addresses real decisions → trust → positions the platform's solutions as the answers.
**Why it worked**: a memorable reframe + a named franchise compounds; pillars let one idea host endless content.
**When to steal**: a named franchise that reframes a shared threat as opportunity, with pillars to sustain it over years.
**Anti-patterns**: doom-framing that paralyses; a "franchise" that's really one report; product push over genuine help.
